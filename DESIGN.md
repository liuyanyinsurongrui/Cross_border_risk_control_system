# DESIGN.md

## 气质与意象
如同海关X光安检仪屏幕般的冷峻精确——深色监控台面上，数据流在暗色背景中发出冷光，每个审核项如同安检传送带上的行李被逐一扫描，违规项以红色警告灯亮起，合规项以绿色通行灯放行。整体气质：高效、冷静、可信赖。

## 视觉策略
- 图形语言：等宽字体数据表格 + 状态指示灯圆点，类似终端/监控面板
- 图像展示：抓取的产品图片以缩略图网格呈现，悬浮放大查看详情，支持AVIF格式
- 图标风格：线条型（Lucide），2px描边，与监控面板气质统一

## 配色方案
- 主背景：深灰近黑 (#0f1117)，模拟监控台暗色环境
- 卡片/面板：#1a1d27，微亮于背景的深色卡片
- 主强调色：青绿 #10b981（合规/通行），警示红 #ef4444（违规/警告）
- 文字主色：#e2e8f0（高对比浅灰白）
- 文字辅助色：#94a3b8（柔和灰色，用于次要信息）
- 进度色：#3b82f6（进行中的蓝色脉冲）

## 字体排版
- 标题/数据：font-sans（PingFang SC / Microsoft YaHei），粗体用于状态标签
- 数据表格：font-mono 用于链接和数字，增强可读性
- 排版节奏：紧凑行距（leading-tight），表格数据密集排列，卡片间留白明确

## 动效与交互
- 审核进度：扫描线动画效果，逐行扫描检查
- 状态切换：简洁的淡入淡出（150ms），不使用弹跳
- 悬停效果：卡片微微亮起边框，缩略图平滑放大
- 违规标记：脉冲红光效果（2s循环），吸引注意但不刺眼

## 页面结构
- 顶部：应用标题栏 + 飞书配置入口 + 审核统计摘要（合规/违规/待审）
- 主体分三栏工作流：
  1. 左侧：数据导入区（上传Excel + 链接列表）
  2. 中部：审核工作台（抓取内容预览 + AI审核分析过程）
  3. 右侧：审核结果面板（结论 + 违规详情 + 推送状态）
- 底部：全局操作栏（一键审核、推送飞书、导出报告）

## 设计禁忌
- 不要使用渐变色背景，保持纯色暗面
- 不要使用圆角大于12px的卡片，保持锐利监控感
- 不要使用暖色调作为主色，保持冷峻专业的科技感
- 不要在数据密集区域使用大间距，紧凑排列优先
- 不要使用卡通或轻松风格的图标/插图
