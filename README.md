# 风控 AI 智能审核系统

面向跨境电商独立站的商品页智能审核工具。  
支持批量导入商品链接、自动抓取页面文字与图片、调用多模态模型进行合规审核，并将审核结果推送到飞书多维表格。

## 核心能力

- 批量导入商品链接，支持 Excel / CSV / 手动粘贴
- 自动识别并抓取商品页文字内容
- 区分产品素材图与详情图，过滤无效小图标与支付图标
- 支持异常页面状态识别，`403/404` 页面不再继续抓取
- 支持多模型配置，按已配置 Key 动态展示可选模型
- 支持 AI 审核、人工复核状态流转
- 支持飞书 OAuth、飞书多维表格推送

## 适用场景

- 跨境电商商品合规初筛
- 商品素材与详情图批量巡检
- 飞书协同审核流程接入
- 独立站商品链接质检与风控

## 系统流程

1. 导入商品链接
2. 抓取页面文字、产品图、详情图
3. 识别异常页面状态码
4. 调用模型执行合规审核
5. 输出审核结论、违规项、分析说明
6. 推送结果到飞书多维表格

## 本地启动

### 1. 安装依赖

```bash
pnpm install
```

### 2. 启动开发环境

```bash
pnpm dev
```

默认访问地址：

```text
http://localhost:5000
```

### 3. 生产构建

```bash
pnpm build
```

### 4. 启动生产服务

```bash
pnpm start
```

## 配置说明

- 模型配置
  - 火山方舟接入点
  - OpenAI Compatible 接口
  - API Key / Base URL / Model Name
- 飞书配置
  - App ID
  - App Secret
  - App Token
  - Table ID
  - OAuth 授权
