# 跨境风控 AI 智能审核系统

面向跨境电商独立站的商品页面审核工具，支持批量导入链接、自动抓取页面文字与图片、调用多模态模型执行合规审核，并将结果回写到飞书多维表格。

## 最近更新

### 2026-07-01

- 新增规则级“双提示词”能力
  - 每条审核规则都可以单独维护 `意图识别提示词`
  - 以及单独维护 `内容审核提示词`
- 审核链路升级为
  - 先做规则意图识别
  - 命中后再执行该规则的内容审核
- 批量审核与网页审核统一走同一套规则意图识别逻辑
- 默认补充了 `禁售商品` 规则的示例提示词

## 核心能力

- 支持 Excel / CSV / 手动粘贴的方式批量导入商品链接
- 自动抓取商品页标题、正文、素材图、详情图
- 自动识别异常页面状态，如 `403` / `404`
- 支持规则级 AI 审核
- 支持规则级意图识别后再进入内容审核
- 支持多模型配置
- 支持审核结果推送到飞书多维表格
- 支持网页审核和后台批量审核两种模式

## 适用场景

- 跨境独立站商品合规初筛
- 禁售商品、侵权、虚假宣传、价格欺诈等风险审核
- 飞书协同审核流程接入
- 商品素材与详情页批量巡检

## 审核流程

1. 导入商品链接
2. 抓取页面标题、正文、商品图、详情图
3. 检查页面状态是否异常
4. 按启用规则执行意图识别
5. 对命中的规则继续执行内容审核
6. 聚合各规则结果，输出最终结论
7. 按需推送到飞书

## 规则配置说明

每条审核规则包含以下字段：

- `规则名称`
  - 仅用于展示和结果标识
- `意图识别提示词`
  - 用于判断该商品是否需要进入当前规则的正式审核
  - 支持占位符：`{product_name}`、`{page_title}`、`{page_url}`、`{text_content}`
- `内容审核提示词`
  - 命中当前规则后，模型按照这段提示词输出标准化审核结果
- `审核模型`
  - 当前规则使用的模型

### 推荐写法

- `意图识别提示词` 只负责“是否值得进入本规则审核”
- `内容审核提示词` 只负责“在本规则下如何判定合规 / 违规 / 待人工复核”
- 意图识别提示词尽量收敛到“命中 / 不命中”，不要直接让模型下最终违规结论

## 飞书配置说明

系统主要通过页面右上角的飞书配置入口完成接入，常见配置项包括：

- `App ID`
- `App Secret`
- `App Token`
- `Table ID`
- `OAuth 授权`

## 当前默认规则

- 虚假宣传
- 知识产权侵权
- 禁售商品
- 价格欺诈
- 误导性图片

其中 `禁售商品` 已内置一版可直接参考的：

- `意图识别提示词` 示例
- `内容审核提示词` 示例

## 注意事项

- 使用前请先完成模型配置，否则无法执行 AI 审核
- 若规则未配置可用模型，系统无法调用该规则审核
- 页面抓取依赖目标站点可访问性，异常页面会直接返回状态说明
- 部分站点如果采用强前端渲染，抓取结果可能受限于原始 HTML 可见内容
